# `ProductCard` - Code Examples with Component Overrides

The `ProductCard` component supports top-level `componentOverrides`, allowing you to customize specific sections without worrying about the other parts. 
Useful when you need to modify a select few parts of the card, without completely replacing the entire component.

## Custom Price Section

<Canvas of={ProductCardStories.CustomPriceSection} />

```tsx
import ProductCard from '@/components/product-card';

const priceOverride = {
  reactNode: (props) => (
    <div className='bg-green-100 p-2 rounded border-l-4 border-green-500'>
      <span className='text-green-800 font-bold text-lg'>
        Special: {props.priceCurrency}{props.product.price}
      </span>
    </div>
  ),
};

function CustomPriceSectionExample() {
  return (
    <ProductCard
      product={{
        id: "highland-golf-pants",
        variationId: "highland-golf-pants--navy",
        name: "Highland Golf Pants",
        imageUrl: "https://example.com/pants.jpg",
        price: "599",
        rating: 4.8,
        reviewsCount: 156,
        description: "Premium golf pants designed for comfort and performance on the course",
      }}
      priceCurrency="$"
      onAddToCart={(e) => console.log('Added to cart', e)}
      componentOverrides={{
        content: {
          price: priceOverride,
        },
      }}
    />
  );
}
```

## Multiple Component Overrides

You can combine multiple component overrides in a single ProductCard:

<Canvas of={ProductCardStories.MultipleCustomizations} />

```tsx
import ProductCard from '@/components/product-card';

function MultipleOverridesExample() {
  return (
    <ProductCard
      product={{
        id: "highland-golf-pants",
        variationId: "highland-golf-pants--navy",
        name: "Highland Golf Pants",
        imageUrl: "https://example.com/pants.jpg",
        price: "799",
        rating: 4.9,
        reviewsCount: 234,
      }}
      priceCurrency="$"
      onAddToCart={(e) => console.log('Added to cart', e)}
      onAddToWishlist={(e) => console.log('Added to wishlist', e)}
      componentOverrides={{
        content: {
          price: priceOverride,
          title: titleOverride,
        },
        footer: {
          addToCartButton: addToCartButtonOverride,
        },
        image: {
          wishlistButton: wishlistButtonOverride,
        },
      }}
    />
  );
}
```
